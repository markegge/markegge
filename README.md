## Mark Egge

**Applied data scientist & transportation planner (AICP).** I turn messy public-sector data into production tools people actually use — interactive transit-planning software, AI pipelines, and nationwide data products.

📍 Bozeman, Montana

### Selected work

- **[GTFS·X](https://gtfsx.com)** — browser-based editor for creating, validating, and exporting standards-compliant GTFS transit feeds (including GTFS-Flex). No GIS software or code required.
- **[CivicCast](https://civiccast.ai)** — turns recorded local-government meetings into AI-generated podcast recaps, delivered to subscribers within 24 hours.
- **[TransitPeers](https://transitpeers.net)** — free peer-benchmarking platform for U.S. transit agencies, built on National Transit Database data · [source](https://github.com/markegge/transitpeers)
- **[`tpm`](https://cran.r-project.org/package=tpm)** — R package on CRAN for FHWA PM3 performance measures (reliability, freight, congestion) from NPMRDS probe data · [source](https://github.com/markegge/fhwa_pm3)
